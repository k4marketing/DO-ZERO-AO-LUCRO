Como gerar layout, textos e imagens com IA — Passo a Passo

  2. Responda: “Qual o tipo de negócio?” (ex: “confeitaria caseira”)

🌐 CAPÍTULO 4 — Hospedando o Site

O que é hospedagem e como funciona?

Hospedagem é o “aluguel de um computador na internet” onde seu site fica armazenado e acessível 24h.

Sem hospedagem, seu site só existe no seu computador — ninguém mais vê.

Passo a passo para usar hospedagens simples

Opção 1: Netlify (grátis para sites estáticos)

Ideal para sites feitos com HTML, CSS, JS (ou exportados do Framer/Wix).

  1. Acesse netlify.com → “Sign Up”
  2. Arraste a pasta do seu site (ou conecte ao GitHub)
  3. Clique em “Deploy site”
  4. Pronto! Seu site ganha um link como: meusite.netlify.app

✅ Grátis, rápido, sem burocracia.

Opção 2: Hostinger (paga, mas completa)

Ideal se for usar WordPress ou quiser domínio próprio desde o início.

  1. Acesse hostinger.com.br
  2. Escolha plano “Single Shared Hosting” (R$ 9,99/mês na promoção)
  3. Cadastre-se e pague
  4. Instale WordPress com 1 clique (ou envie seus arquivos via FTP)
  5. Acesse seu site pelo link temporário

💡 Dica:

Use Hostinger se o cliente quer e-mail profissional (ex: contato@empresa.com.br).

Como enviar os arquivos do site para o servidor

Se você usou Wix/Framer, não precisa enviar arquivos — eles hospedam pra você.

Se fez site do zero (HTML/CSS):

  1. Compacte a pasta do site (index.html, css/, img/) em um .ZIP
  2. No Netlify: arraste o .ZIP na tela inicial
  3. No Hostinger: use o “Gerenciador de Arquivos” ou FTP (FileZilla)

✅ Ferramenta grátis para FTP:

FileZilla (baixe em filezilla-project.org)

✅ Checklist: Escolha sua hospedagem

  • [ ] Site simples, sem WordPress? → Netlify (grátis)
  • [ ] Cliente quer e-mail profissional? → Hostinger (R$ 9,99/mês)
  • [ ] Cliente já tem hospedagem? → Peça acesso FTP e envie os arquivos

🎯 RESUMO RÁPIDO — CAPÍTULO 4

  • Hospedagem é onde seu site “mora” na internet — sem ela, ninguém acessa.
  • Use Netlify para sites estáticos (grátis) ou Hostinger para WordPress (barato).
  • Envie arquivos via arrastar e soltar (Netlify) ou FTP (Hostinger).

🔗 CAPÍTULO 5 — Conectando um Domínio

O que é domínio e DNS?

Domínio = Endereço do site (ex: www.confeitariadonamarta.com.br)
DNS = Sistema que liga o domínio à hospedagem (como um GPS da internet)

Sem configurar o DNS, o domínio não “aponta” para o site.

Onde registrar (exemplos)

Tipo Onde registrar Preço médio/ano
.com.br Registro.br R$ 40,00
.com GoDaddy R$ 60,00
.store / .site Namecheap, Hostinger R$ 20,00

💡 Dica:

Para clientes brasileiros, use .com.br — passa mais credibilidade.

Tutorial: como apontar o domínio para a hospedagem

Exemplo com Registro.br + Netlify

  1. Compre o domínio em registro.br (ex: meusite.com.br)
  2. No painel do Registro.br, vá em “Gerenciar DNS”
  3. Apague todos os registros atuais
  4. Adicione 2 registros do tipo “A”: 
    Nome: @  
Valor: 75.2.63.234 (IP do Netlify — confira no seu painel Netlify)

Nome: www  
Valor: 75.2.63.234
  5. Salve. Pronto!

⏱️ Tempo de propagação:

de 1 minuto a 48 horas (geralmente 5 minutos no Registro.br).

Tempo de propagação do DNS

  • Registro.br: 5 minutos a 1 hora
  • GoDaddy: 1 a 24 horas
  • Namecheap: 30 minutos a 6 horas

✅ Dica:

Depois de configurar, teste em whatsmydns.net — digite o domínio e veja se o IP já está atualizado globalmente.

✅ Checklist: Conexão de domínio

  • [ ] Domínio registrado e pago
  • [ ] Acessou painel DNS do registrador
  • [ ] Apagou registros antigos
  • [ ] Adicionou registros “A” apontando para o IP da hospedagem
  • [ ] Testou em whatsmydns.net

🎯 RESUMO RÁPIDO — CAPÍTULO 5

  • Domínio é o endereço — compre em Registro.br ou GoDaddy.
  • DNS é o “GPS” — aponte para o IP da sua hospedagem.
  • Propagação leva até 48h — mas geralmente é rápido (teste em whatsmydns.net).
